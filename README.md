# واجهة تسجيل دخول فيسبوك التعليمية

هذا مشروع تعليمي يحاكي واجهة تسجيل الدخول إلى فيسبوك لأغراض تعليمية فقط.

## الملفات المضمنة

- `login.html`: ملف HTML الرئيسي الذي يحتوي على هيكل الصفحة
- `styles.css`: ملف CSS الذي يحتوي على تنسيقات الصفحة
- `script.js`: ملف JavaScript الذي يضيف التفاعلية للواجهة

## كيفية الاستخدام

1. افتح ملف `login.html` في متصفح الويب الخاص بك
2. ستظهر لك واجهة تسجيل الدخول المحاكية
3. يمكنك تجربة إدخال بيانات في حقول الإدخال والنقر على الأزرار

## ملاحظات هامة

⚠️ **هذه واجهة تعليمية فقط ولا تقوم بأي عمليات حقيقية:**
- لا يتم إرسال أي بيانات إلى خوادم فيسبوك
- لا يتم تخزين أي بيانات مدخلة
- لا يمكن استخدام هذه الواجهة لتسجيل الدخول الفعلي إلى فيسبوك
- لا يمكن استخدام هذه الواجهة لإنشاء حساب فعلي على فيسبوك

## الاستخدام التعليمي

يمكن استخدام هذه الواجهة للأغراض التعليمية التالية:
- فهم كيفية بناء واجهات تسجيل الدخول
- دراسة تصميم واجهات المستخدم
- تعلم HTML و CSS و JavaScript
- فهم كيفية عمل النماذج في الويب

## تحذير قانوني

هذا المشروع مخصص للأغراض التعليمية فقط. أي استخدام غير مصرح به لهذه الواجهة لأغراض الاحتيال أو الخداع يعتبر غير قانوني وغير أخلاقي.
